# ideviceinstaller-win32

[![Build status](https://ci.appveyor.com/api/projects/status/49lbgi7lwyd5xc82?svg=true)](https://ci.appveyor.com/project/qmfrederik/ideviceinstaller)

Provides a native Windows build (using the Visual C++ compiler) of [ideviceinstaller](http://libimobiledevice.org)

## Where to report issues

For general questions about libimobiledevice, see http://github.com/libimobiledevice/libimobiledevice.
For questions specific to Visual C++, feel free to use the GitHub issue tracker

## How to get the latest binaries

The binaries for ideviceinstaller are available as 
[NuGet CoApp packages](https://www.nuget.org/packages?q=Tags%3A%22libimobiledevice%22).
